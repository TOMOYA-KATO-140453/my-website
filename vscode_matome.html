ショートカットキー 説明
Alt + 矢印 行の移動
Alt + 数字 ページ移動・タブ切替
Ctrl + P ファイル名検索
Ctrl + Shift + P コマンドパレットを開く
Ctrl + W 現在のウィンドウを閉じる
Ctrl + Shift + N 新しいウィンドウを開く
Ctrl + K, Ctrl + S Copilot提案を別枠で表示する※加藤設定
Ctrl + , 設定を開く
Ctrl + J ターミナルを開く/閉じる
Ctrl + Enter 現在の行の下に改行を追加
Ctrl + Shift + Enter 現在の行の上に改行を追加
Ctrl + K → V md プレビュー

ショートカットキー 説明
mkdir ディレクトリ名 新しいディレクトリ（フォルダ）を作成する
cd ディレクトリ名A カレントディレクトリの `ディレクトリ名A` に移動
cd .. 一つ上のディレクトリ（親ディレクトリ）に移動する
cd ..\.. ふたつ上のディレクトリ（親ディレクトリ）に移動する
cd .\一つ下の階層のディレクトリ名\ 一つ下の階層の指定したディレクトリ名に移動
cd C:\Users\140453\Documents 絶対Pathで移動
.\venv\Scripts\Activate pythonの仮想環境を起動
deactivate pythonの仮想環境を終了
npm start 開発サーバーを起動してReactアプリケーションを実行
Ctrl + C 実行中のサーバープロセスが停止
